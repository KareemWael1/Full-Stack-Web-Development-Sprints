Notes:
Sentences written in .html file will be displayed in browser.
Tags in general must be closed with slash if there is content written within it (<tag> content </tag>).
Search engines uses <meta name="description" content="webpage description"> to find relevant webpages.
Text get smaller from <h1>Title</h1> to <h6>Text</h6>
Comment: <!--comment-->

Tags:
<html>  </html>     : Identify the file as .html
<head>  </head>     : Data related to webpage (name in tabs bar, icon, etc..)
<body>  </body>     : All webpage contents
<!DOCTYPE html>     : Identify the file as .html 5 version
<meta  >            : Store data
<h1>Title</h1>      : Largest text
<h2>Text</h2>       : Larger text
<h3>Text</h3>       : Large text
<h4>Text</h4>       : Small text
<h5>Text</h5>       : Smaller text
<h6>Text</h6>       : Smallest text
<p>Paragraph</p>    : Paragraph
<hr>                : Horizontal row (_________)
<style>  </style>   : Adding css code
<script>  </script> : Adding JavaScript code


Head tags:
<title>  </title> : Sets webpage name in tabs bar

<meta attributes>:
name = "name of data"
content = "value of data"
